# hema_FinOMOP_federated
Federated descriptive analysis of common hematological cancers

This repo contains information, files and scripts for running  federated readiness and analytical scripts on selected common hematological cancer within the FinOMOP network. Contact person is [Mikael Csonka](mailto:Mikael.Csonka@helsinki.fi), a MD PhD student in the iCAN-SHARE group in Helsinki, and [Kimmo Porkka](mailto:kimmo.porkka@helsinki.fi), the PI. 

For performing the validation, please see the [readiness_script](./readiness_script) and [analysis_script](./analysis_script) folders for details. A short protocol for IRB submissions can be found at [protocol](./protocol) folder.

# hema_FinOMOP_federated
Federated descriptive analysis of common hematological cancers

This repo contains information, files and scripts for running  federated readiness and analytical scripts on selected common hematological cancer within the FinOMOP network. by [Mikael Csonka](mailto:Mikael.Csonka@helsinki.fi), a MD PhD student in the iCAN-SHARE group in Helsinki. 

For performing the validation, please see the [readiness_package](./readiness_package) and [analysis_package](./analysis_package) folders for details. A short protocol for IRB submissions can be found at [ms_comments_protocol](./ms_comments_protocol).
